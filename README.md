# homebrew-tap

[Homebrew](https://brew.sh/) tap for [@tonkotsuboy](https://github.com/tonkotsuboy)'s tools.

## Formulae

### [pr-approve](https://github.com/tonkotsuboy/pr-approve)

Approve GitHub PRs from your terminal or Raycast, with pre-flight safety checks.

```bash
brew install tonkotsuboy/tap/pr-approve
```
